# 全局量化交易核心规则（永久生效）

## 1. TQSDK 开发强制规范

- 代码必须使用 Python 3.13 + 完整类型注解
- 异步 async/await 调用 TQSDK 接口，分离配置/策略/数据模块
- 禁止硬编码密钥，必须添加异常处理、空值校验
- 订单执行必须包含重试、超时、风控拦截逻辑

## 2. 回测校验铁律

- 严禁未来函数、幸存者偏差、数据断层
- 成交价格使用下一根K线开盘价，滑点/手续费参数真实
- 严格校验夏普比率、最大回撤、胜率等风险指标

## 3. 期货交易风控合规

- 单品种仓位≤账户资金20%，杠杆不超交易所限制
- 禁止高频报单/无效价格报单，遵守交易所交易时段
- 极端行情、网络中断必须有应急暂停/强平保护
